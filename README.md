# DOTFILES

## Setup
- Clone the repository
- Run `./install.sh`
- Run `/stow-init.sh`
- In vim, run `:Copilot setup`

## Shortcuts
### Quickfix List
- Next: `:cnext` or `<C-j>`
- Previous: `:cprev` or `<C-k>`
- Open: `:copen`
- Close: `:cclose` or `:ccl`

## Github Copilot
- Setup: `:Copilot setup`
- Help: `:help copilot`
